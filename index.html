⚙️ Definições
🔑 Chave API
Usa a tua chave API-Football Pro para aceder aos dados de odds e estatísticas.
🎯 Casa de Apostas Preferida
Se apostas sempre na mesma casa (ex: Betclic em Portugal), escolhe-a aqui. A app vai usar as odds dessa casa em vez de mostrar sempre a melhor odd global (que podes não conseguir usar).
📤 Página Pública (Afiliação)
Exporta as picks do dia para um ficheiro JSON. Faz upload para o GitHub Pages e qualquer pessoa pode ver as tuas tips numa página pública só-leitura.
Casas legais em Portugal · cola os links de afiliado:
Como usar:
1. Cola os links de afiliado das casas que tens parceria e clica Guardar
2. Clica em "Exportar picks.json" → faz upload do ficheiro para o GitHub
3. Clica em "Descarregar public.html" (só uma vez) → faz upload para o GitHub
4. Partilha o link github.io/teu-repo/public.html
Os botões só aparecem na página pública para casas que tiverem link configurado.
🔧 Cache e Diagnóstico
Limpa os dados em cache e força um novo carregamento. Usa o diagnóstico para ver o estado das APIs.

📤 Exportar Histórico
